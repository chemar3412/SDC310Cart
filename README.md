# SDC310Cart
##Project Description
This project is a simple webpage development that connects to a SQL database. allowing user to add items to cart where the tax and shipping cost is calculated and the total cost. Allows the user to change the quantity of each item in the cart. 

Project Tasks
Task 1: Set up the Database for development
Task 2: SQL database is connected the the .php file
Task 3: Database support added, Create, Update, Remove added into the coding.
Task 4: MVC implemented into .php file
Task 5: Testing of the code and final edits are done.
